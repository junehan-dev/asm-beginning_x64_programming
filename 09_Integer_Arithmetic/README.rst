9. Integer Arithmetic
=====================

Integer arithmetic Instructions
-------------------------------

- ``add``
   ADD   DEST, SRC
- ``sub``
   SUB   DEST, SRC
- ``inc``
   INC   DEST
- ``dec``
   DEC   DEST
- ``sal``\: Shift arithmetic left
   SAL   DEST, COUNT
- ``sar``\: Shift arithmetic right
- ``imul``\: Signed Multiplication
   - IMUL SIZE SRC
   IMUL SIZE SRC
- ``mul``\: Unsigned Multiplication
- ``idiv``\: Signed Division
- ``div``\: Unsigned Division
